# 03 - Mi producto

## Qué es
Sistema completo para hamburgueserías: web de pedidos online + CRM + herramientas de fidelización y gestión. Todo integrado, a medida, con diferenciales reales.

## Funcionalidades core
- **Catálogo online** con configuraciones (medallones, aderezos, extras, salsas)
- **Carrito + checkout**: delivery o retiro, métodos de pago múltiples, descuento efectivo
- **Microturnos dinámicos**: el cliente elige un horario, el sistema controla la capacidad automáticamente
- **Tracking en tiempo real**: el cliente ve el estado de su pedido sin preguntar
- **Panel de admin completo**: pedidos, productos, categorías, cupones, turnos
- **Vista de cocina**: pantalla dedicada para el cocinero, sin ruido extra
- **Sistema de fidelización** ("soles" / "figuritas"): acumulación automática, canje, aprobación
- **Reseñas de productos**: con moderación, imágenes, reportes
- **Reseñas Google**: prompt automático post-entrega + email con anti-spam
- **Notificaciones in-app**
- **Perfil de usuario + direcciones guardadas**
- **Login con Google** (OAuth)

## Por qué es mejor que las alternativas
- No es un template: es a medida con identidad del negocio
- Fidelización real integrada (casi nadie lo tiene)
- Tracking en tiempo real = menos atención al cliente = menos estrés
- El hamburguesero no necesita aprender nada técnico
